reciprocalspaceship.algorithms.merge¶

reciprocalspaceship.algorithms.merge(dataset, intensity_key='I', sigma_key='SIGI', sort=False)[source]¶

Merge dataset using inverse-variance weights.

Parameters

  • dataset (rs.DataSet) – Unmerged DataSet containing scaled intensities and uncertainties

  • intensity_key (str) – Column name for intensities

  • sigma_key (str) – Column name for uncertainties

  • sort (bool) – If True, index of returned DataSet will be sorted

Returns

rs.DataSet – Merged DataSet object
